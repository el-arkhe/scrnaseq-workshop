# Scripts del taller

Los scripts en este directorio están numerados y deben ejecutarse en orden.

Cada script corresponde a un paso del flujo de análisis de `scRNA-seq`.
Los scripts contienen comentarios básicos; la explicación conceptual se
proporciona durante la sesión y en los documentos dentro de `docs/`.

Orden recomendado de ejecución:

1. 01_load_data.R
2. 02_qc_metrics.R
3. 03_filter_cells.R
4. 04_normalization.R
5. 05_clustering.R
